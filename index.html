📱 Guide d'installation MaghrebTV

Créer l'APK Android avec Android Studio · Guide pas à pas

✅ Projet Android Studio prêt → MaghrebTV-AndroidStudio.zip

Contient tout le code source · À ouvrir dans Android Studio
ÉTAPE 1

Installer Java (JDK)

Android Studio a besoin de Java pour fonctionner. Télécharge et installe :

https://adoptium.net → Télécharger JDK 17 LTS

Installe-le normalement (Suivant → Suivant → Terminer).

💡 Si Java est déjà installé sur ton PC, passe directement à l'étape 2.
ÉTAPE 2

Installer Android Studio

Télécharge Android Studio (gratuit, officiel Google) :

https://developer.android.com/studio
  • Clique Download Android Studio
  • Lance l'installeur → accepte les options par défaut
  • Au premier démarrage, laisse Android Studio télécharger le SDK Android (peut prendre 10–20 min)
💡 Espace requis : environ 8 Go. Assure-toi d'avoir de la place sur ton disque.
ÉTAPE 3

Ouvrir le projet MaghrebTV

  • Dézippe le fichier MaghrebTV-AndroidStudio.zip sur ton bureau
  • Dans Android Studio → clique Open (ou File → Open)
  • Navigue jusqu'au dossier MaghrebTV dézippé
  • Clique OK

Android Studio va analyser le projet et télécharger les dépendances (1–3 min).

💡 Tu verras une barre de progression en bas. Attends qu'elle disparaisse avant de continuer.
ÉTAPE 4

Générer l'APK

Dans le menu Android Studio :

  • Clique Build (menu en haut)
  • Puis Build Bundle(s) / APK(s)
  • Puis Build APK(s)

Attends 2–5 minutes. Une notification apparaît en bas :

✅ "APK(s) generated successfully" → Clique "locate" pour trouver le fichier APK.

L'APK se trouve dans :

MaghrebTV/app/build/outputs/apk/debug/app-debug.apk
ÉTAPE 5

Installer l'APK sur ton téléphone Android

Option A — Via câble USB

  • Branche ton téléphone au PC
  • Sur le téléphone → Paramètres → Options développeur → Débogage USB (activer)
  • Dans Android Studio → clique le bouton ▶ vert en haut → sélectionne ton téléphone
  • L'app s'installe et se lance automatiquement 🎉

Option B — Copier l'APK directement

  • Copie le fichier app-debug.apk sur ton téléphone (WhatsApp, email, USB…)
  • Sur le téléphone → ouvre le fichier APK
  • Autorise "Sources inconnues" si demandé → Installer
⚠️ Si ton téléphone bloque l'installation : Paramètres → Sécurité → Activer "Installer des apps inconnues"
BONUS

Activer les options développeur sur Android

Pour activer le débogage USB (Option A) :

  • Paramètres → À propos du téléphone
  • Clique 7 fois sur "Numéro de build"
  • Les Options développeur apparaissent dans Paramètres
  • Active Débogage USB
💡 Fonctionne sur Samsung, Xiaomi, Huawei, OnePlus… la manipulation est identique.
